# @cobalt/docs

Documentation site for the Cobalt Design System.

VitePress site that hosts component demos, the design token reference, the icon browser, and contributor / designer / developer guides. Reads tokens, components, and icons live from sibling packages — the docs site is the canonical home for usage, theming, and API content for the rest of the monorepo.

> **Internal package — not published.** Run from a clone of the monorepo (`pnpm dev`).
