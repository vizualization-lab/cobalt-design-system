# @cobalt/vue

Vue 3 wrappers for the Cobalt Design System.

Defined with `defineComponent` so each Cobalt web component ships with type-safe props and event forwarding for Vue templates. Re-exports the components from `@cobalt/components` — install this package only if you want the Vue-flavored API; the underlying web components work in Vue directly as well.

**Peer dependencies:** Vue 3.4+.

## Install

```bash
npm install @cobalt/vue
```

See the Cobalt docs site for component examples, prop tables, and event handling patterns.
