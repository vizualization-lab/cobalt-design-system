# @cobalt/workbench

Component development environment for the Cobalt Design System.

Lightweight Vite app used while authoring or debugging components. Each component has a preview file under `previews/` that mounts the component with a few representative configurations. Used for quick visual iteration without spinning up the full docs site.

> **Internal package — not published.** Run from a clone of the monorepo (`pnpm workbench`).
