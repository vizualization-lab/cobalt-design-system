# @cobalt/react

React wrappers for the Cobalt Design System.

Generated with `@lit/react` so each Cobalt web component ships with native React props and event bindings. Re-exports the components from `@cobalt/components` — install this package only if you want the React-flavored API; otherwise the underlying web components work in React directly.

**Peer dependencies:** React 18 or 19.

## Install

```bash
npm install @cobalt/react
```

See the Cobalt docs site for component examples, prop tables, and event handling patterns.
