# @cobalt/angular

Angular wrappers for the Cobalt Design System.

Provides Angular directives that synchronize properties and forward custom events from the underlying Cobalt web components. Re-exports the components from `@cobalt/components` — install this package only if you want the Angular-flavored API; the underlying web components also work in Angular templates with `CUSTOM_ELEMENTS_SCHEMA`.

**Peer dependencies:** Angular 17, 18, 19, 20, or 21.

## Install

```bash
npm install @cobalt/angular
```

See the Cobalt docs site for component examples, directive APIs, and event handling patterns.
