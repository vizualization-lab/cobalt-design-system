Designer Resources ​

Helpful links, tools, and references for designers working with the Cobalt Design System.

Note: This page is a living document. Content will be expanded as new tools and guides become available.

Figma Library ​

  • Cobalt Components — The production-ready component library. Enable it via Assets > Team Library in any Figma file.
  • Cobalt Icons — The full icon set, available as a separate library.
  • Cobalt Tokens — Color, spacing, and typography styles synced with the codebase.

For setup instructions, see the Getting Started for Designers guide.

Recommended Figma Plugins ​

  • Cobalt Design System — Inspect token values, check contrast ratios, and sync with the latest token updates
  • Contrast — Quick WCAG contrast ratio checks for any color pairing
  • Able — Accessibility annotations and focus order documentation
  • Tokens Studio - Load, update, and export tokens managed by Cobalt

Handoff Workflow ​

A smooth handoff reduces back-and-forth between designers and developers. Key resources:

  • Figma Handoff Guide — What to include when handing off designs (includes a copy-paste template)
  • Token Structure — Understanding the three-tier token architecture
